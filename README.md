# readme-ai-doctor 🩺

A tool that scores your GitHub README out of 100 and suggests improvements using AI.

Paste your README, get a score, and see exactly what's missing.

**Live Demo:** [readme-ai-doctor.vercel.app](https://readme-ai-doctor-srqs.vercel.app)

---

## What it checks

- Installation section
- Usage section
- Contributing guidelines
- License
- Badges
- Content length

## Tech Stack

- React (Vite)
- Gemini AI API
- React Router
- CSS

## Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/readme-ai-doctor.git
cd readme-ai-doctor
npm install
```

Create a `.env` file:
```
VITE_GEMINI_API_KEY=your_gemini_key_here
```

```bash
npm run dev
```
